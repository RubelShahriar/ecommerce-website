## GadgetsPro

Live Website: https://gadgetsproeco.vercel.app/

```
• Here it is a GadgetsPro ecommerce application developed with next js
• A user can add product to cart and can increase or decrese quantity of a product
• After clicking a product it navigates user to the product details page from where user can buy it
• Payment system integreted with stripe and a user can complete a payment while buying a product
• Users can use this website in smartphone, tablet and laptops as it is responsive
```
